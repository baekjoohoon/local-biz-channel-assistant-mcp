# 동네 장사 카톡 비서 MCP

AGENTIC PLAYER 10 출품을 목표로 만든 Remote MCP 서버입니다. 소상공인이 카카오톡 채널에서 바로 사용할 수 있는 캠페인, 고객 응대, 메뉴 구성, 주간 실행계획을 생성합니다.

## 왜 이 아이템인가

카카오톡 채널은 많은 소상공인이 이미 고객과 만나는 접점입니다. 하지만 대부분은 어떤 메시지를 언제 보내야 하는지, 불만 응대나 재방문 유도를 어떻게 해야 하는지 매번 새로 고민합니다.

이 MCP 서버는 매장 정보를 입력받아 카카오톡 채널에서 실행 가능한 형태로 바꿉니다.

- 동네 매장 상황 진단
- 카카오톡 채널 캠페인 생성
- 고객 답장 초안 생성
- 채널용 메뉴 구성 점검
- 7일 성장 실행계획 생성

## 실행

```bash
npm install
npm run dev
```

기본 서버 주소:

```text
http://localhost:3000
```

MCP endpoint:

```text
http://localhost:3000/mcp
```

## 배포

Docker 기반 배포를 지원합니다.

```bash
docker build -t dongne-biz-channel-assistant .
docker run -p 3000:3000 dongne-biz-channel-assistant
```

카카오 PlayMCP 등록 시에는 배포된 HTTPS 주소의 `/mcp` endpoint를 등록합니다.

## 도구 목록

### diagnose_local_shop

매장 정보와 현재 고민을 바탕으로 포지셔닝, 강점, 위험 신호, 다음 행동을 제안합니다.

### create_channel_campaign

캠페인 목표, 기간, 예산 수준에 맞춰 카카오톡 채널 메시지, 게시 일정, KPI, 주의사항을 생성합니다.

### draft_customer_reply

예약, 불만, 가격 문의, 품절, 지연, 리뷰 감사 등 고객 응대 문구를 생성합니다.

### review_menu_for_channel

메뉴 목록을 분석해 대표 메뉴, 세트 구성, 추가 판매 후보, 메뉴판 개선 포인트를 제안합니다.

### build_weekly_growth_plan

이번 주 목표와 사용 가능한 시간을 기준으로 7일 실행계획을 만듭니다.

## 제출 포지셔닝

서비스명 후보:

```text
동네 장사 카톡 비서
```

한 줄 소개:

```text
소상공인이 카카오톡 채널에서 바로 쓸 수 있는 캠페인과 고객 응대를 만들어주는 AI 장사 비서입니다.
```

차별점:

- 카카오톡 채널이라는 실제 고객 접점에 맞춘 결과물을 생성합니다.
- 소상공인이 바로 복사해 쓸 수 있는 짧은 문구와 실행계획을 제공합니다.
- 외부 API 없이도 안정적으로 작동하며, 향후 Kakao Tools 위젯과 결합하기 쉽습니다.

심사 기준 대응:

- 창의성: MCP를 단순 정보 조회가 아니라 동네 매장 운영 액션으로 연결합니다.
- 편의성: 사장님이 질문 한 번으로 메시지, 혜택, 일정, KPI까지 얻습니다.
- 안정성: 입력 스키마가 명확하고 네트워크/API 키 의존성이 없습니다.

## 사용자가 직접 해야 할 일

계정 권한이 필요한 아래 작업은 사용자가 직접 진행해야 합니다.

- 카카오 계정 로그인
- 카카오클라우드 MCP 서버 생성
- PlayMCP 개발자 콘솔 등록
- 심사 요청 및 전체 공개 전환
- 공모전 페이지에서 최종 예선 참여 버튼 클릭
